Welcome to my Final Project.

In this project, I will be doing basic registration to demonstrate cookies, and serving different versions of a page to a user depending on whether they are an admin or a user.


If they are an admin, they will:

  • Have access to previously registered users, whose values are stored in an array based on what number they are.
  • Be able to "ban" or delete those users
  • Be able to manipulate the user's username and password
  • Be able to do whatever a user is able to do

If they are a user, they will:

  • Be able to deposit money, which adds it to their total balance
  • Be able to withdraw money, which subtracts it from their total balance
